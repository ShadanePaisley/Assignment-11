# Module 11 Assignment

This repository contains my work for Module 11. The assignment involved creating meaningful visualizations and dashboards using Matplotlib to analyze retail sales data for SunCoast Retail, a multi-location company across Florida.

## Files Included
- module11-assignment.py

## What I Practiced

In this assignment, I practiced creating and customizing diverse chart types (line charts, bar charts, scatter plots, histograms, box plots, and pie charts) using Matplotlib. I learned to integrate Pandas for data grouping, aggregation, and transformation across multiple dimensions (location, category, time period). I applied design principles like colour coding, annotations, reference lines, and professional formatting to enhance clarity. I also practiced selecting appropriate visualizations for different analytical questions, computing derived metrics like efficiency ratios and correlation lines, and translating visual insights into data-driven business recommendations for stakeholders.
